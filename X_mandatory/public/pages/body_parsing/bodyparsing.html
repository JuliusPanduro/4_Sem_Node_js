Why body parsing?

Body parsing is used so we can work with the correct format of the data.

In the pictures to the right, it is showed how to implement body parsing in Express, and how we can work with it in an REST api.

The first pictures shows that we use the app.use(), to set up / telling the server that we are going to body parse in JSON formats and thats why we are using express.json().

The second picture shows us how we can work with the body, based on the clients requests, and since we already defined that we are using JSON formats with the picture one, we dont have to parse anything at this moment.
